      Panel layout
      
Rearrange the profile panels
        and resize the plot. There are 5 panels; rows are filled
        left-to-right, top-to-bottom. Rows auto-grow if you don't give
        enough for the chosen columns.

      

        
        
        
        
        
      

    


    

      Axis ranges
      
Type real values (e.g. 1e5); leave a
        box blank to autoscale that end. Or just use the mouse on the
        plot: drag to zoom, double-click to reset.
